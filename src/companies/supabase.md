## Company blurb

[Supabase](https://supabase.com/) is an open-source backend-as-a-service that bundles a managed Postgres database with authentication, storage, edge functions, vector embeddings, and realtime subscriptions. It is widely positioned as the open-source alternative to Firebase, and its core building blocks are themselves open-source projects layered on top of Postgres.

Supabase is a [Y Combinator alum](https://www.ycombinator.com/companies/supabase) (S20) and has grown into one of the most popular developer platforms for shipping full-stack apps quickly.

## Company size

Approximately 150-200 employees.

## Remote status

Supabase is a "born-remote" company. The team is distributed across more than 50 countries and has no central headquarters. Roles are typically tagged for AMER, EMEA, or APAC time-zone overlap.

## Region

Worldwide. Supabase hires across the Americas, Europe, and Asia-Pacific.

## Company technologies

- Postgres at the core
- TypeScript and Deno for edge functions and SDKs
- Go and Rust for performance-critical services
- Kong for the API gateway
- Self-hostable open-source stack alongside the managed cloud
