## Company blurb

[Bitwarden](https://bitwarden.com/) is an open-source password and secrets management platform for individuals, teams, and enterprises. The product spans password management, passwordless authentication, secrets management for developers and infrastructure, and self-hostable server deployments.

Bitwarden's clients and server are open source on [GitHub](https://github.com/bitwarden), and the company has been independently audited multiple times.

## Company size

Approximately 200-300 employees.

## Remote status

Bitwarden is a fully distributed company. The team works from the United States, EMEA, ANZ, and additional regions, with worldwide remote roles regularly posted.

## Region

Worldwide.

## Company technologies

- C# / .NET for the server
- TypeScript and Angular for the web vault and browser extension
- Native mobile clients for iOS and Android
- Rust for performance-critical and shared cryptographic components
- Azure for hosting the managed service
