## Company blurb

[Temporal](https://temporal.io/) is the company behind the open-source [Temporal](https://github.com/temporalio/temporal) durable execution platform. Temporal makes it possible to write long-running, fault-tolerant workflows as ordinary code, with built-in retries, state persistence, and exactly-once semantics. The company offers Temporal Cloud, a managed version of the platform, alongside the open-source server and SDKs.

Temporal is used in production by companies including Snap, Stripe, Datadog, Box, and Netflix for orchestrating mission-critical workflows.

## Company size

Approximately 300-400 employees.

## Remote status

Temporal is a remote-first company with flexible schedules. The team is distributed across the U.S., Canada, U.K., Germany, India, and Singapore, with access to coworking space via WeWork for those who want it.

## Region

Worldwide.

## Company technologies

- Go for the Temporal server
- SDKs in Go, Java, TypeScript, Python, .NET, Ruby, and PHP
- Kubernetes for managed cloud deployments
- Cassandra and PostgreSQL for persistence
- Multi-cloud (AWS, GCP) for Temporal Cloud
