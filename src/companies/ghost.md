## Company blurb

[Ghost](https://ghost.org/) is an open-source publishing platform for independent creators, journalists, and publishers. It combines a modern editor, membership and subscription tooling, newsletters, and a JAMstack-friendly headless API into a single product, available either self-hosted or via [Ghost(Pro)](https://ghost.org/pricing/).

Ghost is run by a non-profit foundation and has powered publications for organizations including Mozilla, OpenAI, and many independent newsletters.

## Company size

Approximately 30-50 employees.

## Remote status

Ghost has been a fully remote company since its founding in 2013. The team is intentionally kept small (around 50 people) and is spread across many countries, with twice-yearly in-person team retreats.

## Region

Worldwide.

## Company technologies

- Node.js for the core platform
- Ember.js for the admin client
- MySQL as the primary database
- A headless Content API for use with static site generators and frontends
- Stripe-based subscriptions and payments
