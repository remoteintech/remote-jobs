## Company blurb

[RevenueCat](https://www.revenuecat.com/) is the subscription management infrastructure used by a large share of mobile apps with in-app purchases. It provides cross-platform SDKs (iOS, Android, web, React Native, Flutter, Unity), a backend that normalises Apple, Google, Stripe, and Amazon billing, plus analytics, experiments, and paywall tooling.

The company is widely used by independent developers and large publishers alike, and it publishes the influential [State of Subscription Apps](https://www.revenuecat.com/state-of-subscription-apps/) report each year.

## Company size

Approximately 100-200 employees.

## Remote status

RevenueCat has been remote-first since day one. The team is distributed across more than 25 countries, with explicit async-first communication and location-agnostic compensation.

## Region

Worldwide.

## Company technologies

- Go for backend services
- Python for data and ML pipelines
- Native mobile SDKs (Swift, Kotlin) plus cross-platform SDKs
- TypeScript and React for the dashboard
- AWS for infrastructure
