## Company blurb

[Hugging Face](https://huggingface.co/) is the leading open platform for machine learning. It hosts more than a million models, datasets, and Spaces (interactive ML demos) and maintains widely used open-source libraries including [Transformers](https://github.com/huggingface/transformers), [Diffusers](https://github.com/huggingface/diffusers), [Datasets](https://github.com/huggingface/datasets), and [PEFT](https://github.com/huggingface/peft).

Beyond the Hub, Hugging Face offers Inference Endpoints, AutoTrain, and enterprise plans, and it collaborates broadly with the open-source ML ecosystem.

## Company size

Approximately 250+ employees.

## Remote status

Hugging Face is a remote-first organization with team members in 30+ countries. The company has optional offices in cities such as Paris, New York, Miami, London, and Bern, but most roles can be filled remotely.

## Region

Worldwide.

## Company technologies

- Python for the ML stack (PyTorch, JAX, Transformers, Diffusers)
- Rust for performance-critical components (e.g. tokenizers, safetensors)
- TypeScript and React for the Hub frontend
- Cloud-hosted inference on AWS and GCP
- Open-source-first — most code is developed in public on GitHub
