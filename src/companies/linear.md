## Company blurb

[Linear](https://linear.app/) is a project and issue tracking tool built for high-performance software teams. It is known for its speed, opinionated workflow, real-time collaboration, and offline-first sync engine. Linear is used by tens of thousands of product and engineering teams, from early-stage startups to large public companies.

The company publishes the [Linear Method](https://linear.app/method), a public set of practices for building software products that has become widely cited in the industry.

## Company size

Approximately 100-150 employees.

## Remote status

Linear is a fully distributed company. Hiring is open across the Americas, Europe, and Asia-Pacific, and the team is built around asynchronous collaboration with periodic in-person offsites.

## Region

Worldwide.

## Company technologies

- TypeScript across the stack
- React for the web client
- GraphQL with a custom sync engine
- Node.js for backend services
- Postgres and Redis
- AWS for infrastructure
